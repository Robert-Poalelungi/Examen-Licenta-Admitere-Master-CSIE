# Examen Licență & Admitere Master — CSIE ASE București

Repo cu materiale pentru **examenul de licență CSIE** și **admiterea la masteratele CSIE** la ASE București.

---

## Structura repo

```
📁 Examen Licenta/
│   📄 README.md                          ← teorie completă pentru examenul de licență
│   📄 Tematica-si-bibliografia-...pdf    ← tematica oficială licență 2025-2026
│   📁 Carti/                             ← cărțile de referință (PDF)
│   └── 📁 intrebari/                     ← întrebări practice, quiz, documente pregătire
│
└── 📁 Admitere Master/
    │   📄 README.md                      ← ghid complet admitere master (calendar, locuri,
    │                                        taxe, documente, tematică, bibliografie)
    │   📄 Tematica_si_bibliografie_...pdf ← tematica oficială examen specialitate 2026
    │
    ├── 📁 CSIE_3/                        ← cod tematică concurs CSIE 3
    │   │                                    (Informatică economică, E-Business,
    │   │                                     Baze de date, SIMRPE, TIID)
    │   │   📄 README.md                  ← teorie: BD, SQL, PL/SQL, C, POO C++
    │   ├── 📁 Subiecte/
    │   │       📄 README.md              ← rezolvări subiecte examen 2019–2024
    │   │       📄 CSIE3_20XX.pdf         ← subiecte originale (2019–2024)
    │   └── 📁 Carti/                     ← 5 cărți de referință (PDF)
    │
    └── 📁 CSIE_4/                        ← cod tematică concurs CSIE 4
        │                                    (Securitatea informatică — în engleză)
        │   📄 README.md                  ← teorie: C (K&R), C++ (Stroustrup), Java (OCP SE 17)
        ├── 📁 Subiecte/
        │       📄 README.md              ← rezolvări subiecte examen 2019–2024
        │       📄 CSIE4_20XX.pdf         ← subiecte originale (2019–2024)
        └── 📁 Carti/                     ← 3 cărți de referință (PDF)
```
